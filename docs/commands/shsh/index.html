shsh

Dump shsh blob

Setup

Jailbreak your iDevice and install openssh

❯ brew install libusbmuxd

❯ iproxy 2222 22

Dumping shsh blobs allows you to downgrade iOS later.

❯ ipsw shsh

      • Parsing shsh
      • Parsing IMG4
