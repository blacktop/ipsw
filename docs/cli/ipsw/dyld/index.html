ipsw dyld​

Parse dyld_shared_cache

ipsw dyld [flags]

Options​

  -h, --help   help for dyld

Options inherited from parent commands​

      --color           colorize output
      --config string   config file (default is $HOME/.ipsw/config.yaml)
  -V, --verbose         verbose output
