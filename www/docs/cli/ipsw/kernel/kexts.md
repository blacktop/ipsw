## ipsw kernel kexts

List kernel extensions

```
ipsw kernel kexts <kernelcache> [flags]
```

### Options

```
  -a, --arch string   Which architecture to use for fat/universal MachO
  -d, --diff          Diff two kernel's kexts
  -h, --help          help for kexts
  -j, --json          Output kexts as JSON
```
