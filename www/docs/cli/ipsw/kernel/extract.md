## ipsw kernel extract

Extract KEXT(s) from kernelcache

```
ipsw kernel extract <KERNELCACHE> <KEXT> [flags]
```

### Options

```
  -a, --all             Extract all KEXTs
  -h, --help            help for extract
      --output string   Directory to extract KEXTs to
```
