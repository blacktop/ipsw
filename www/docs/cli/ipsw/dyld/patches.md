## ipsw dyld patches

Dump dyld patch info

```
ipsw dyld patches <dyld_shared_cache> [flags]
```

### Options

```
  -h, --help           help for patches
  -i, --image string   dylib image to search
  -s, --sym string     dylib image symbol to dump patches for
```
