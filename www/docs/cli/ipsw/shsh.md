## ipsw shsh

Get shsh blobs from device

```
ipsw shsh [flags]
```

### Options

```

  -t, --host string   ssh host (default "localhost")
  -n, --insecure      ignore known_hosts key checking
  -i, --key string    ssh key (default "$HOME/.ssh/id_rsa")
  -p, --port string   ssh port (default "2222")
```
